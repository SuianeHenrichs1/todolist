## ✨ Projeto

O TodoList permite aos usuários:

- 📋 Adicionar tarefas
- ✏️ Editar tarefas existentes
- 🗑️ Remover tarefas
- ✅ Marcar tarefas como concluídas
- 🎨 Design responsivo, funcionando bem em todos os dispositivos
